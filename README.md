# raytracer

C++ project

Graphical renderer that uses vector math to simulate light ray projections.

This project was built in order to better understand computer graphics. 
Built on pure C++.

To build, use open project in visual studio or use GCC.

gcc *.cpp -o Output

### Future Changes
